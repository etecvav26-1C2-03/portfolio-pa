# Exercicio 2

#### "Um programa deve receber três números e mostrar qual deles é o maior."

### Algoritmo do problema em linguagem natural:

- 1-Ler o primeiro número
- 2-Ler o segundo número
- 3-Ler o terceiro número
- 4-Comparar os três números
- 5-Verificar qual deles é o maior
- 6-Mostrar o maior número

# Exercicio 4
 
  As IDEs (como o Portugol Studio), bibliotecas e frameworks funcionam como uma "caixa de ferramentas" pronta. Elas são importantes porque 
permitem que o programador foque na solução do problema (a lógica de negócio) em vez de perder tempo com tarefas repetitivas ou excessivamente 
técnicas, como configurar a comunicação com o hardware ou desenhar cada pixel de uma interface.

 ### Quais problemas surgiriam ao desenvolver tudo do zero?
	Aumento drástico de erros (Bugs): Criar componentes complexos do zero aumenta a chance de falhas de segurança e lógica.
	Inviabilidade de tempo: O que hoje leva dias para ser feito levaria anos, tornando o software muito caro.
	Dificuldade de manutenção: Sem um padrão (framework), cada programador escreveria de um jeito, tornando quase impossível para outra pessoa entender o código depois.
	Justificativa com Exemplos (Contexto Portugol)
	
  ### Exemplo de Biblioteca (inclua biblioteca Matematica):
	No Portugol, se você precisar calcular a raiz quadrada de um número, você usa a função Matematica.raiz().
	Sem a ferramenta: Você teria que implementar manualmente algoritmos matemáticos complexos (como o método de Newton) antes mesmo de começar o seu programa.
	Com a ferramenta: Você apenas chama a função e foca no resultado.
	
  ### Exemplo de IDE (Portugol Studio):
	A IDE ajuda com o realce de sintaxe (cores no código) e mensagens de erro em tempo real.
	Sem a ferramenta: Se você esquecesse um fechar chaves } ou um ponto e vírgula em um editor de texto simples (como o Bloco de Notas), 
	teria que ler linha por linha no escuro para achar o erro.
	Com a ferramenta: A IDE aponta exatamente a linha onde a sintaxe está incorreta, economizando horas de depuração.
	
   ### Exemplo de Framework (Estruturas de Dados):
	Imagine ter que programar como o computador organiza a memória para cada variável toda vez que criar uma lista. 
	Ferramentas que já trazem "vetores" prontos evitam que o programador precise lidar com endereços de memória física, 
	o que seria extremamente difícil para iniciantes.
